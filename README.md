# Dice_game
# 🎲 Dice Game

A simple and interactive Dice Game built using **HTML, CSS, and JavaScript**. This project generates random dice rolls for two players and automatically determines the winner each time the page is refreshed.

## 🚀 Live Demo

🔗 https://izza-faris.github.io/Dice_game/

## 📌 Features

✨ Random dice generation on every page refresh

🎲 Two-player dice competition

🏆 Automatic winner announcement

🔄 Draw detection when both players get the same number

📱 Responsive and clean user interface

---

## 🛠️ Technologies Used

- HTML5
- CSS3
- JavaScript (ES6)

🎮 How It Works
1.The page loads.
2.JavaScript generates two random numbers between 1 and 6.
3.Each number is assigned to a dice image.
4.The dice images are displayed.
5.The game compares both values.
6.A winner or draw message is shown automatically.
